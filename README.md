# Visualização

Atividade de visualização utilizando P5 realizada em sala. A proposta foi de desenvolver uma microinterface interativa pensando no universo do projeto do módulo, tornando a solução mais viva, compreensível e memorável. 

Mais informações sobre o proejto podem ser visualizadas na pasta visualização, que contem toda a documentação e código da atividade.
